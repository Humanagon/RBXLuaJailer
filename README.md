# RBXLuaJailer
## Welcome to the RBXLuaJailer repository!
RBXLuaJailer is a security/modding wrapper for the ROBLOX Instance class API that allows you to whitelist only the members you want to be used, as well as even add your own.

Here's a video on basic usage of the module: https://www.youtube.com/watch?v=eQEIJra1Mkg
